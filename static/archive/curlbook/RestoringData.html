NOTE:  To use the following method of restoring data, you MUST have updated to the most recent version of CurlBook.


Rolling CurlBook back to an earlier state (to restore data from an earlier time) is accomplished through iTunes.  First, connect your iPhone, iPod Touch, or iPad to your Mac or PC, and open iTunes.  Select your device in the left side-bar:




At the top of the screen, select the ‘Apps’ tab, and scroll down to ‘File-Sharing’:




When you select CurlBook, you will see a number of files:





The “Season.cbseason” file is your current season.  Each ‘.cbbackup’ file is an archive created every time you save a game.  The name of the file corresponds to the date and time of the file’s creation.  For example, the top file in the image above is named 201010311203.cbbackup.


The string of numbers is broken down as follows:  2010-10-31 12:03, or 12:03 on October 31, 2010.


To restore any backup, simply change the extension of the file from ‘.cbbackup’ to ‘.cbrestore’:


DO NOT CHANGE THE FILE NAME, ONLY THE EXTENSION.  IF THE STRING OF NUMBERS CHANGES, THE FILE WILL NOT LOAD.




If your device supports fast-app-switching (iPhone 4/3GS, iPod Touch 4G/3G running iOS 4 or above), you must ensure CurlBook is performing a fresh launch.


To ensure this, while not running CurlBook, double-click the home-button to bring up the fast-app-switching tray, press and hold on the CurlBook icon, and tap the minus that appears:




This ensures CurlBook is not loading from the background, but will perform a fresh launch.


If double-clicking the home button does not bring up the fast-app-switching tray (shown above), your device does not support multitasking, and this is not an issue.


Now you can launch CurlBook, the backup file will be read, and your data will be restored.


Pro Tip: you can use this method to transfer a season from one device to another.  Simply copy the Season.cbseason file from one device to another (through iTunes file-sharing), or copy a .cbbackup file, and use this method to load it.
