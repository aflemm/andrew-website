First, tap the ‘+’ in the upper right corner of the ‘Teams’ tab:




Once here, the four players of the team must be selected.  This is accomplished by tapping on a position, and selecting the player:




Tap on the player you would like to set for that position.  If the player you need has not been created, tap the ‘+’ in the upper right corner to create a new player.  For information on creating new players, see ‘Creating a player’.


If you do not know the name of a player, a ‘filler-player’ can be created.  To create filler-players, simply leave a position blank (do not select a player, and do not create a player).  When you tap ‘Save’, the following prompt will be shown:




Tap ‘Continue’ to create however many filler-players you have left blank.  For more information on filler-players, be sure to read ‘Filler player info’.


The ‘Team Name’ field is automatically populated with the skip’s name.  To edit the default name, simply tap and change:




Select whether the team is ‘ours’, or ‘theirs’.  This is merely a convenience, which will separate ‘our’ teams from ‘their’ teams in the ‘Teams’ tab, and when creating a game.




Once the name has been set, and all players have been chosen (unless filler players are desired), tap the ‘Save’ button to save the new team:




Once the team is saved, it will appear in the ‘Teams’ tab.  The number beside the team name indicates how many games the team has played.
