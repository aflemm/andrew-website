Tabs


Any data stored in CurlBook can be accessed from any of the main tabs (Games,  Players, Teams, Clubs, Events). 


For example, a game could be opened in all 5 tabs:

  1. •In ‘Games’, select the game you are interested in.

  2. •In ‘Players’, select a player who played in the game, tap ‘Games’ to see all of that player’s games, and select the game.

  3. •In ‘Teams’, select a team that played in the game, tap ‘Games’ to see all of that team’s games, and select the game.

  4. •In ‘Clubs’, select the club at which the game was played, tap ‘Games’ to see all of the games played at that club, and select the game.

  5. •In ‘Events’, select the event which the game was part of, tap ‘Games’ to see all of the games played in that event, and select the game.


This can be useful in many ways:

  1. •Opening a game, for example, in multiple tabs can help quick switching between scoring the game, and viewing the stats. 

  2. •At the end of the game, a stats-view for each each player can be opened in each tab, allowing for quick debriefing.

  3. •Game notes can be kept open in one tab, while the game is being scored in another.  This saves navigating to the top-level of the game to create each note.  Along a similar line, team notes can be kept open in yet another tab.


Battery Life


CurlBook itself is not a battery-intensive application, however to be able to stat 3 games on a single charge, it is recommended that you take the following precautions:


Turn off wireless

Turn down screen brightness

Refrain from listening to music

Put display to sleep between shots/ends whenever possible
