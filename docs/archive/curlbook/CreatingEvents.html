From the ‘Events’ tab, tap the ‘+’ button in the upper right hand corner:



First, fill in the ‘Event Name’ field with the name of the event:




Next, the main club of the event must be set.  Tap ‘Main Club’:




If the club you want has already been created, tap to select it.  To see more information on any club, tap the blue arrow beside its name.


If the club has yet to be created, tap the ‘+’ in the top right corner to create a new club.  For information on creating a new club, see ‘Creating a club’.


Once the club and name are set, the new event can be saved.  Tap the ‘Save’ button in the top right corner:




Upon tapping ‘Save’, the new event will immediately appear in the ‘Events’ tab:
