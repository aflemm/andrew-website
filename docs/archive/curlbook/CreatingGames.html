From the ‘Games’ tab, tap on the ‘+’ button.




In the ‘New game’ view, tap ‘Event’




If the event you want has already been created, tap to select it.  To see more information on any event, tap the blue arrow beside its name.


If the event has yet to be created, tap the ‘+’ in the top right corner to create a new event.  For information on creating a new event, see ‘Creating an event’.


Once the event has been selected, a ‘Game description’ field is added.  This is an optional ‘name’ for the game.  ie) ‘Semi-final’, ‘Game 1’, etc.  Leaving this field blank will have no ill-effects.


Next, the teams must be chosen.  Tap on ‘Our team’:




If the team you want has already been created, tap to select it.  To see more information on any team, tap the blue arrow beside its name.


If the team has yet to be created, tap the ‘+’ in the top right corner to create a new team.  For information on creating a new team, see ‘Creating a team’.


To select ‘their’ team, follow the same procedure.


Selecting ‘our’ team adds an ‘Our line-up’ field.  This feature allows the game to be played with fewer than 4 players, with the skip not throwing the last rock, or with a substitute player.  By default, the line-up is set as ‘Standard’.  To play with a standard line-up, do nothing.  For information on setting an alternate line-up, see ‘Custom Lineup Info’.


Once the teams and line-up are chosen, the club must be set.  By default, the club is set to the main club of the chosen event.  To alter this (for events taking place at more than one club), tap ‘Club’:




If the club you want has already been created, tap to select it.  To see more information on any club, tap the blue arrow beside its name.


If the club has yet to be created, tap the ‘+’ in the top right corner to create a new club.  For information on creating a new club, see ‘Creating a club’.


Once a club is selected, a ‘Sheet’ field is added.  To select the sheet ,tap ‘Sheet’:




Tap the desired sheet to select it.


Once a sheet has been chosen a ‘Colour’ field is added.  Tap ‘Colour’ to select ‘our’ set of rocks:




Tap ‘our’ team’s colour to select.


Once the colour is selected, the ‘Save’ button will become active.  Enough information has been added to save the game, however there are three remaining fields, each containing important information.


The time/date fields are set to the current time/date by default.  To change one, tap it:






Select the proper date/time, and tap ‘Save’ to complete.


The final step is to input whether we have the hammer in the first end.  It is optimistically  set as ‘Yes’.  To change, simply tap on the field, and it will become ‘No’:




Now that all of the information has been entered, tap ‘Save’ to save the game.  Keep in mind that only some of these properties can be altered once the game is saved:



With that, you will have created a new game, which will appear immediately in the ‘Games’ tab.  The green ‘W’ indicates that our team is not losing.
